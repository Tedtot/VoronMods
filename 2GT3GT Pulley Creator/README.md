# 2GT/3GT Pulley Creator

## Information
Generates a 2GT/3GT pulley based on Gates spec. Use Fusion parameters to change the # of teeth, bore diameter, belt pitch, and pulley length.

Note the pulley does not include a hub nor flanges.
